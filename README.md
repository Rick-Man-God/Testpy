# Creates XEL wallet vanity addresses

## Install python 
```
sudo apt-get update
```
```
sudo apt-get install python3.6
```
## Install python pip 

```
sudo apt install python3-pip
```
## Make xel.py executible 

```
sudo chmod +x xel.py
```

## Run it 

```
sudo ./xel.py
```

## Change Prefix 

```
poss = ['1337', '1337-H4X0R']
```
Change to what you want to have for example KING
